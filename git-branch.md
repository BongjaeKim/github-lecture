## Git Branching
* 브랜칭 Branching?
	- 메인 라인에서 새로 브랜치(가지)를 내서 새 브랜치에서 작업을 수행하고 변경 이력을 남김.
	- 이 새 브랜치에서 수행한 작업은 메인 라인에 영향을 주지 않음.
* 브랜칭은 VCS의 꽃!
* Git의 장점 중의 하나가 가볍고 빠른 브랜칭
